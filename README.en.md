# Soundroom

A music web application for real-time collaborative listening.  
Users can join rooms and control playback synchronously.

## Tech Stack
- Frontend: Next.js, React, TypeScript, GraphQL (`graphql-request`), WebSocket
- Backend: NestJS, GraphQL, WebSocket, PostgreSQL, TypeORM
